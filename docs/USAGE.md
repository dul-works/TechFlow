# TechFlow — 사용법

미디어아트 행사용 하드웨어 시스템 구성 문서(System Configuration, Cable Guide, Elec & Network, Hardware List)를 웹에서 제작하고 PPT로 내보내는 도구입니다.

---

## 시작하기

```bash
cd app
npm install        # 최초 1회
npx vite --port 5173
```

브라우저에서 `http://localhost:5173` 접속

---

## 화면 구성

| 영역 | 역할 |
|------|------|
| 좌측 사이드바 (상단) | 페이지 이동: Cover / System Config / Cable Guide / Elec & Network / Hardware List |
| 좌측 사이드바 (하단) | Zone(존) 관리 |
| 상단 바 (우측) | JSON 저장·불러오기, PPT 내보내기 |
| 중앙 메인 영역 | 현재 페이지 편집 화면 |

---

## 1. Cover 페이지

- 우측 패널에서 **프로젝트명**과 **날짜**를 입력합니다.
- 중앙 미리보기에서 최종 표지 디자인을 확인할 수 있습니다.

---

## 2. Zone(존) 관리

하나의 행사에 여러 존(Zone)이 있을 경우, 각 존을 독립적으로 관리합니다.

- **System Config / Cable Guide / Elec & Network / Hardware List** 페이지에서 좌측 사이드바에 Zone 목록이 표시됩니다.
- `+ New Zone` 버튼으로 새 존 추가, 이름을 **더블클릭**하면 이름 변경 가능합니다.
- 존 이름을 클릭하면 해당 존의 편집 화면이 열립니다.
- 존을 삭제하면 해당 존의 모든 데이터(블럭, 케이블, 도면, 마커)가 함께 삭제됩니다.

---

## 3. System Configuration 페이지

행사 내 하드웨어 구성과 케이블 연결을 다이어그램으로 제작합니다.

### 하드웨어 블럭 배치

- **좌측 팔레트**에서 하드웨어를 **클릭** 또는 캔버스로 **드래그**해 배치합니다.
- 검색창 및 카테고리 탭(Display / PC / Network 등)으로 빠르게 찾을 수 있습니다.
- 목록에 없는 장비는 팔레트 하단 `+ Custom Hardware`로 추가합니다.
- 모든 블럭은 **20px 그리드에 자동 스냅**됩니다.

### 케이블 연결

1. **상단 케이블 선택 바**에서 케이블 종류(HDMI / UTP / USB 등)를 먼저 선택합니다.
2. 하드웨어 블럭에 마우스를 올리면 연결 핸들(●)이 나타납니다.
3. 핸들을 드래그해 다른 블럭의 핸들에 연결합니다.
4. 연결 후 케이블을 클릭하면 우측 패널에서 종류·색상을 변경할 수 있습니다.

### 블럭/케이블 편집

- 블럭 또는 케이블을 클릭하면 **우측 패널 하단**에 인라인 편집기가 열립니다.
  - 블럭: 이름, 모델, 수량, 담당(LOCAL / I M FINE) 변경
  - 케이블: 종류, 색상 변경
- `Delete` 키 또는 블럭 우측 상단 **×** 버튼으로 삭제합니다.

### Technical List 자동 생성

- 우측 패널에 배치된 하드웨어와 케이블 목록이 **자동**으로 표시됩니다.
- 색상 사각형을 클릭하면 색상 변경 팝업이 열립니다.
- 이 목록은 PPT 출력 시 오른쪽 패널로 그대로 반영됩니다.

---

## 4. Cable Guide 페이지

실제 도면/현장 이미지 위에 하드웨어 위치와 케이블 연결을 표시합니다.

### 도면 이미지 업로드

1. 좌측 `📁 Upload Image` 버튼 → 이미지 파일 선택
2. 이미지 편집 팝업에서:
   - **드래그**로 크롭 영역 지정
   - **투명도 슬라이더**로 배경 밝기 조절 (권장: 30~60%)
   - **흑백 전환 버튼**으로 Grayscale 변환
3. `Apply` 버튼으로 적용합니다.

### 하드웨어 블럭 위치 지정

- System Config에 배치된 하드웨어가 캔버스에 자동으로 표시됩니다.
- 각 블럭을 **드래그**해 실제 현장 위치에 맞게 이동합니다.
- 케이블 연결선은 System Config의 정보를 기반으로 **자동 표시**됩니다.

---

## 5. Electricity & Network 페이지

전원 콘센트와 인터넷 설치 위치를 도면 위에 표시합니다.

### 도면 공유

- Cable Guide와 **동일한 도면 이미지**가 사용됩니다.
- 이미지 설정(투명도/흑백)도 공유됩니다.

### 마커 배치

1. 좌측 패널에서 마커 종류를 클릭합니다:
   - ⚡ **220V** — 전원 콘센트
   - 🌐 **Internet** — 유선 인터넷
   - 🔌 **Powerstrip** — 멀티탭
   - 🔗 **USB Repeater** — USB 연장 장치
2. 캔버스를 클릭하면 해당 위치에 마커가 배치됩니다.
3. 배치된 마커는 **드래그**로 이동, **×** 버튼으로 삭제합니다.

---

## 6. Hardware List 페이지

System Config에서 배치한 하드웨어가 자동으로 표에 집계됩니다.

- **LOCAL 탭**: 현장/운영사가 준비해야 할 항목
- **I M FINE 탭**: 내가 준비해야 할 항목
- 행의 `→ I M FINE` / `→ Local` 버튼으로 담당을 전환합니다.
- **Remark** 셀을 클릭하면 비고를 직접 입력할 수 있습니다.
- 담당 변경은 System Config 블럭에도 즉시 반영됩니다.

---

## 7. 저장 & 불러오기

| 버튼 | 동작 |
|------|------|
| `📂 Load JSON` | 저장된 `.json` 프로젝트 파일을 불러와 이어서 편집합니다 |
| `💾 Save JSON` | 현재 프로젝트 전체 상태를 `.json`으로 저장합니다 |

- JSON에는 모든 블럭 위치, 케이블 연결, 도면 이미지, 마커 정보가 저장됩니다.
- 저장한 JSON을 다시 불러오면 웹에서 그대로 이어서 편집할 수 있습니다.

---

## 8. PPT 내보내기

상단 `📊 Export PPT` 버튼을 누르면 `.pptx` 파일이 자동 다운로드됩니다.

### 출력 슬라이드 구성 (Zone당)

1. 프로젝트 표지
2. Zone 표지
3. System Configuration
4. Cable Guide
5. Electricity & Network
6. Hardware List (LOCAL)
7. Hardware List (I M FINE)

### 폰트 설치 안내

PPT에는 **Samsung SS Head KR** / **Samsung SS Body KR** 폰트가 지정됩니다.  
올바르게 표시되려면 해당 폰트가 시스템에 **설치**되어 있어야 합니다.

- 폰트 파일 위치: 프로젝트 루트 `Fonts/` 폴더의 `.ttf` 파일
- 설치 방법: `.ttf` 파일을 **더블클릭** → 설치 버튼 클릭

### PPT 내 편집 방식

PPT 출력물은 PowerPoint에서 모든 요소를 직접 편집할 수 있도록 제작됩니다.

| 요소 | PPT 형식 | 편집 방법 |
|------|---------|---------|
| 하드웨어 블럭 | 사각형 안 텍스트 (단일 도형) | 더블클릭 → 텍스트 편집 |
| 케이블 연결선 | bentConnector3 / straightConnector1 | 클릭 → 색상/굵기 변경 |
| 하드웨어 리스트 | PowerPoint 표 (addTable) | 셀 클릭 → 텍스트 편집 |
| 색상 바 (블럭 상단) | 별도 rect 도형 | 클릭 → 채우기 색상 변경 |
